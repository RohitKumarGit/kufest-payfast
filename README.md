# kufest-payfast
You can directly view the app at : https://kufest.herokuapp.com/

Steps To run Locally :

Clone the repo and move to the directory

## Install all packages
```
npm install
```

### Start the front End
```
npm run serve
```
### move to back end folder
```
cd backend
```

### Start Back end server
```
node app.js
```
